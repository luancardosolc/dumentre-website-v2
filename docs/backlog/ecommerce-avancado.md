# Backlog: E-commerce e Lojas Virtuais Avançadas 🛒

## Objetivo
Estruturar a oferta comercial de lojas virtuais completas (e-commerce) na Dumentre, delimitando escopos técnicos e alinhamentos de expectativas.

## Por que isso importa comercialmente?
E-commerce é uma das maiores demandas de pequenas empresas, mas costuma apresentar alta complexidade operacional (integração com Correios/Logística, Gateways de Pagamento, Nota Fiscal, emissão de estoques). Ter uma oferta bem estruturada evita retrabalho e desalinhamentos de prazos e valores.

## Escopo Inicial
- Definição de tecnologias sugeridas (ex.: Shopify para e-commerce rápido, WooCommerce para flexibilidade ou customizado via APIs seguras para alta escala).
- Criação de material explicativo destacando a necessidade de mapeamento de frete, impostos e cadastro de produtos.
- Linkagem da solução de e-commerce como capacidade avançada no site institucional.

## Fora de Escopo por enquanto
- Desenvolvimento de e-commerce em fase inicial sem reuniões de alinhamento prévio profunda.

## Critérios de Aceite
- Apresentação transparente dos requisitos de um e-commerce no FAQ do site.
- Material de prospecção pronto focado em alinhar prazos com o cliente.

## Ideias Futuras
- Criação de uma solução headless e-commerce unindo Next.js/Vite com API Shopify para lojas de performance ultra-rápida.
