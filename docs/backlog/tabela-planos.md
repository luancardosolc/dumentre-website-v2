# Backlog: Tabela de Planos e Assinaturas 💰

## Objetivo
Avaliar e criar uma seção ou página de comparação de escopos de planos comerciais estruturados para facilitar a pré-venda.

## Por que isso importa comercialmente?
Muitos clientes gostam de ver faixas de preço ou entregáveis consolidados para entender a ordem de grandeza do investimento necessário. Uma tabela clara de pacotes ajuda na autodecisão do cliente, filtrando leads que já possuem o orçamento necessário.

## Escopo Inicial
- Estruturação de 3 planos com foco comercial:
  - **Plano Start:** Soluções de entrada para negócios locais (SEO local, página de orçamento simples).
  - **Plano Pro:** Landing page de alta conversão + chatbot básico + CRM starter.
  - **Plano Growth:** Sistemas web customizados, integrações avançadas e manutenção mensal robusta.
- Exibir os benefícios de cada um sem valores numéricos explícitos inicialmente, estimulando o clique para solicitar orçamento.

## Fora de Escopo por enquanto
- Exibição de preços em moeda local sem prévia consulta comercial.

## Critérios de Aceite
- Tabela responsiva de fácil leitura em dispositivos móveis (empilhamento de planos).
- Destaque visual no plano mais popular/recomendado.

## Ideias Futuras
- Calculadora de orçamento interativa onde o cliente marca as features que precisa e vê uma estimativa de preço dinâmica.
