# Backlog: Integração com Sistemas de CRM 🎛️

## Objetivo
Configurar a integração automática dos pontos de contato do site com sistemas de CRM (Customer Relationship Management) para a gestão comercial.

## Por que isso importa comercialmente?
Centralizar e organizar os leads que chegam é vital para evitar perda de oportunidades comerciais. Ao enviar contatos de formulário ou interações diretamente para o funil de vendas do CRM, a Dumentre garante atendimento ágil e histórico completo dos clientes prospectados.

## Escopo Inicial
- Mapeamento das APIs de CRMs Starter de mercado (ex. Kommo, Bitrix24, HubSpot ou RD Station CRM).
- Desenvolvimento de webhook para envio de dados do lead capturado no formulário do site para o CRM escolhido.

## Fora de Escopo por enquanto
- Desenvolvimento de CRM próprio da Dumentre (utilizaremos ferramentas consolidadas de mercado para os nossos clientes e para nossa própria operação).

## Critérios de Aceite
- Dados de novos contatos chegando corretamente no funil do CRM selecionado em menos de 1 minuto.
- Notificação automática do time comercial por e-mail ou WhatsApp a cada novo lead integrado.

## Ideias Futuras
- Distribuição inteligente de leads automática (round-robin) caso o time comercial cresça no futuro.
