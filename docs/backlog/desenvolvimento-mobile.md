# Backlog: Desenvolvimento Mobile 📱

## Objetivo
Estruturar e disponibilizar o serviço de desenvolvimento de aplicativos móveis sob medida para plataformas iOS e Android.

## Por que isso importa comercialmente?
Muitos negócios atingem um nível de maturidade digital onde um site ou bot de WhatsApp não é suficiente. Eles precisam de engajamento recorrente, recursos nativos do aparelho (GPS, câmera, Bluetooth), notificações instantâneas e uma experiência premium dedicada para fidelização de clientes. A oferta de desenvolvimento mobile posiciona a Dumentre como uma software house completa e premium, aumentando consideravelmente o ticket médio dos projetos.

## Escopo Inicial
- **Framework:** Utilização de **React Native** com **Expo** para desenvolvimento multiplataforma (iOS e Android) ágil e eficiente.
- **Funcionalidades Core:**
  - Telas e fluxos responsivos com UI/UX premium.
  - Integração segura com APIs REST / GraphQL da Dumentre ou de terceiros.
  - Sistema de **Autenticação** robusto (JWT, OAuth2, Login Social).
  - Configuração e envio de **Push Notifications** (via Firebase Cloud Messaging / Expo Notifications).
- **Publicação:** Processo estruturado para submissão e publicação nas lojas oficiais (Google Play Store e Apple App Store).
- **Observabilidade & Analytics:**
  - Integração nativa com **Firebase Crashlytics** e **Sentry** para monitoramento proativo de erros.
  - Eventos de comportamento de uso com **Firebase Analytics**.

## Fora de Escopo por enquanto
- Desenvolvimento puramente nativo (Swift/Kotlin nativos separados).
- Jogos mobile complexos em 3D.
- Integração com hardware de nicho extremamente específico sem SDKs de mercado.

## Critérios de Aceite
- Apresentar a capacidade de entrega de aplicativos mobile na copy comercial do site oficial da Dumentre de forma nítida.
- Disponibilizar um fluxo claro na captação de leads para clientes que buscam especificamente desenvolvimento de aplicativos.
- Ter a base de boilerplate e documentação técnica pronta em React Native/Expo para agilizar a criação do primeiro projeto.

## Ideias Futuras
- Criação de um produto SaaS de aplicativo de catálogo padrão para pequenos comércios a um custo mensal acessível.
- Integração profunda com inteligência artificial para rodar modelos locais leves (On-Device AI) no celular.
